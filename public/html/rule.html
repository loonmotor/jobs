Rules

Here are a few rules to get you started in no time

Once you have signed up / signed in, you will have access to the following use cases :

  • You can create a profile. A profile is required if you would like to submit your interest to a job.
  • You can create a company. A company is required if you would like to create a job.
  • You can create a job. Make sure you have created at least one company beforehand.
  • You can edit your profile.
  • You can edit and/or remove your companies. Removing a company will remove all its associated jobs too.
  • You can edit and/or remove the jobs offered by your companies.
